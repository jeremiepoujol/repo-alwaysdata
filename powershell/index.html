Utiliser PowerShell pour interagir avec l'API GitHub

Il est possible d'utiliser PowerShell pour automatiser la gestion de vos dépôts GitHub. Cela inclut :

  • Créer, modifier ou supprimer des fichiers dans un dépôt GitHub
  • Créer et gérer des GitHub Pages (puisque ce sont des fichiers dans une branche dédiée)
  • Automatiser les commits et les pushes de contenu web statique

Exemple de requête API avec PowerShell

$token = 'ghp_votreTokenIci'
$headers = @{
    Authorization = "token $token"
    Accept        = "application/vnd.github.v3+json"
}
$repo = "utilisateur/nom-du-repo"
$url = "https://api.github.com/repos/$repo/contents/"

Invoke-RestMethod -Uri $url -Headers $headers -Method Get

Étapes pour démarrer

  1. Créer un Personal Access Token (PAT) sur GitHub avec les permissions nécessaires.
  2. Installer PowerShell sur votre machine si ce n'est pas déjà fait.
  3. Utiliser PowerShell pour envoyer des requêtes vers l'API GitHub et gérer votre contenu.

Pour aller plus loin, pensez à utiliser le module PowerShellForGitHub pour simplifier les appels API !
