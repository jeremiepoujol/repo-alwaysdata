Comprendre les boucles for en JavaScript

La boucle for permet de répéter une série d'instructions plusieurs fois. Elle est composée de trois parties :

  • Initialisation : on commence par définir la valeur de départ de notre compteur (var i = 0;).
  • Condition : tant que cette condition est vraie, la boucle continue (i < n;).
  • Incrémentation : à chaque passage, on augmente ou diminue notre compteur (i++; signifie i = i + 1;).

Exemple simple :

for (var i = 0; i < 5; i++) {
    console.log(i);
}

Cette boucle affichera : 0, 1, 2, 3, 4.

Important : On peut imbriquer des boucles pour créer des effets plus complexes comme des formes avec des caractères : un triangle ou un carré d'étoiles ⭐️.
